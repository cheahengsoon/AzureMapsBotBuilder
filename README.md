# Integrate Azure Maps with Bot Builder v4
**Use Case** <br/>
Get Arrival Time of the Bus Id via Bus Stop in Penang, Malaysia.

## Steps
1. Get Transit Route
2. Get Transit Itinerary
3. Get Real Time Arrivals
